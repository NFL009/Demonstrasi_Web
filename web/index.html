Q : Emang manfaatnya apa ya?

A : Banyak manfaatnya, terutama untuk pengembangan diri:

Hard Skills:

  • Bisa menguasai Bahasa Pemrograman
  • Paham Algoritma dan Struktur Data
  • Tahu cara mengembangkan aplikasi dan sistem
Soft Skills:
  • Makin bisa problem solving
  • Makin jago berpikir analitis dan logis
  • Menambah kreativitas dan jago berinovasi

Q : Software atau alat apa aja yang dipakai?

A : Tergantung materi, bisa pakai Visual Studio Code, Python, C++, JavaScript, Arduino IDE, dan GitHub.

Q : Berapa kali pertemuan per minggu?

A : Biasanya 1 kali seminggu, tapi kalau ada event atau project bisa lebih sering.
